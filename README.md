A sleek AI chatbot powered by **Google Gemini 2.0 Flash** — built with pure HTML, CSS & JavaScript. No frameworks. No installation.

---

## Features
- Real-time AI responses with conversation memory
- Markdown rendering — code blocks, bold, lists & more
- Secure API key handling — stored in browser, never in code
- Clean dark UI with smooth animations
- Works instantly — just open in any browser

---

## Quick Start
1. Get a free API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Open `cognix-chatbot.html` in your browser
3. Paste your key → click **Save Key** → start chatting

---

## Tech Stack
`HTML5` `CSS3` `JavaScript` `Gemini 2.0 Flash API`
